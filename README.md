# Productivity App
## This is a Productivity application that combinea core productivity features into a single, lightweight web application.

## Key Features (V1)
- Pomodoro timer to support focused work sessions
- To-do list with task creation and completion tracking
- Calendar to add and view important tasks or events
- Light and dark theme toggle

## Planned Enhancements
- User authentication and personalisation
- Notifications and reminders 
- Analytics on sessions

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **Deployment:** (to be added)

## Learning Objectives
- Build a full-stack application with clean separation between frontend and backend
- Design REST APIs for task and timer management
- Work with state management and asynchronous API calls in React
- Deploy a production-ready web application
